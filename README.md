1. Descrição do Projeto
Este repositório contém uma Análise Exploratória de Dados (EDA) desenvolvida para a disciplina de Ciência de Dados. O objetivo foi analisar o perfil dos beneficiários do PROUNI no ano de 2020, utilizando técnicas estatísticas e visualizações em Python.

2. Sobre o Dataset
Fonte: https://dados.gov.br/dados/conjuntos-dados/mec-prouni
Registros: 166.830 linhas e 15 colunas.
Variáveis Principais: Idade, Raça, Sexo, Região e Tipo de Bolsa.

Observação: Devido ao limite de 25MB do GitHub para upload, o dataset original foi compactado. Para rodar o notebook localmente, é necessário extrair o arquivo ProuniRelatorioDadosAbertos2020.zip na mesma pasta do projeto.

3. Análises e Insights Obtidos

Perfil Jovem: A análise da Moda e Mediana confirma que a grande maioria dos bolsistas são jovens entre 18 e 22 anos.

Inclusão de Adultos: O Boxplot revelou diversos outliers na faixa acima dos 40 anos, evidenciando que o programa também serve como porta de entrada para requalificação profissional tardia ou para pessoas que buscam fazer outras graduações.

Equilíbrio Regional: Os gráficos de dispersão indicam que o perfil etário é consistente em todas as regiões do Brasil, mostrando que o programa atinge seu público-alvo de forma homogênea.
